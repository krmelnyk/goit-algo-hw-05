# Порівняння алгоритмів пошуку підрядка

Мета роботи - порівняти швидкість трьох алгоритмів пошуку підрядка на двох текстових файлах:

- Кнута-Морріса-Пратта (KMP)
- Рабіна-Карпа (Rabin-Karp)
- Боєра-Мура (Boyer-Moore)

Вимірювання проводилось за допомогою timeit (100 запусків).

Для перевірки використано два підрядки:

- існуючий підрядок: `алгоритм`
- вигаданий підрядок: `superoptimizedquantumsearch`

## Результати

Результати наведено як середній час одного запуску в секундах.

| Алгоритм | Стаття 1, існуючий | Стаття 1, вигаданий | Стаття 2, існуючий | Стаття 2, вигаданий |
|---|---:|---:|---:|---:|
| KMP | 0.000024 | 0.001176 | 0.000325 | 0.001526 |
| Rabin-Karp | 0.000039 | 0.002205 | 0.000521 | 0.003146 |

## Висновки для кожного тексту

Для `стаття 1.txt` найшвидшим алгоритмом став Boyer-Moore. Він показав найменший час як для підрядка, що існує в тексті, так і для вигаданого підрядка.

Для `стаття 2.txt` найшвидшим алгоритмом також став Boyer-Moore. Особливо помітна перевага цього алгоритму при пошуку вигаданого підрядка, оскільки він швидко пропускає частини тексту завдяки таблиці зміщень.

## Загальний висновок

Boyer-Moore показав найкращий результат у всіх тестових сценаріях, тому для цих двох текстів він є найефективнішим алгоритмом пошуку підрядка.

KMP працює стабільно, але в цих вимірюваннях поступається Boyer-Moore. Rabin-Karp виявився найповільнішим серед трьох алгоритмів через додаткові операції обчислення та перевірки хешів.
