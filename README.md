# Порівняння алгоритмів пошуку підрядка

Було реалізовано алгоритми:
- KMP
- Rabin-Karp
- Boyer-Moore

Вимірювання проводилось за допомогою timeit (100 запусків).

## Результати

У більшості випадків Boyer-Moore показав найкращу швидкодію,
особливо при пошуку підрядка, який не існує в тексті.

KMP демонструє стабільний час виконання незалежно від структури тексту.

Rabin-Karp показує хороші результати, але поступається
через додаткові хеш-перевірки.

## Загальний висновок

Boyer-Moore є найефективнішим алгоритмом для великих текстів,
особливо у випадках відсутності підрядка.
